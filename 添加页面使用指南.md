# 添加页面功能使用指南

## 1. 登录后台管理系统

1. 打开浏览器，访问后台管理地址：http://localhost:5000/admin.html
2. 在登录界面输入密码：`admin123`（默认密码，可以在admin.js中修改）
3. 点击"登录"按钮

## 2. 进入页面管理

1. 登录成功后，您将看到管理界面
2. 点击顶部的"页面管理"标签页

## 3. 添加新页面

1. 在页面管理界面中，找到并点击"添加新页面"按钮
2. 弹出第一个模态框：
   - 输入您想要创建的页面名称（例如："新闻"、"资源下载"等）
   - 点击"确认"按钮
3. 弹出第二个模态框（页面类型选择）：
   - 选择页面类型：
     - **文章类页面**：类似首页，适合展示文章列表
     - **资源类页面**：适合展示资源下载列表
     - **游戏类页面**：适合展示游戏相关内容
     - **空白页面**：适合自定义内容
   - 点击"确认"按钮

## 4. 查看和编辑新页面

1. 创建成功后，系统会自动切换到新创建的页面
2. 在页面选择器中可以看到您刚刚创建的页面
3. 您可以编辑页面标题、内容等信息
4. 点击"保存页面"按钮保存更改

## 5. 查看新页面在前端的显示

1. 创建的页面会自动添加到导航菜单中
2. 访问首页：http://localhost:5000
3. 在导航菜单中可以看到新添加的页面链接
4. 点击链接可以访问新创建的页面

## 注意事项

1. 页面名称不能为空
2. 页面类型必须从四个选项中选择一个
3. 创建页面后会自动添加到导航菜单
4. 所有页面更改都需要点击"保存页面"按钮才能生效

## 常见问题

**Q: 为什么创建页面后导航菜单中没有显示？**
A: 请检查是否点击了"保存页面"按钮，或者刷新一下浏览器页面。

**Q: 如何删除不需要的页面？**
A: 目前系统不支持直接删除页面，您可以联系管理员进行手动删除。

**Q: 页面类型可以修改吗？**
A: 目前系统不支持修改页面类型，创建时请谨慎选择。
